# Wrapper Engine

## Описание

Проект реализует обёртку (`Wrapper`) над методами любых классов с произвольной сигнатурой и движок (`Engine`) для вызова этих методов по имени.  
Поддерживаются методы с любым количеством аргументов, с возвращаемым значением или `void`.  
Используется `std::any` для хранения аргументов и `Holder` для хранения результата вызова.

Проект также включает тесты на GoogleTest для проверки работы обёртки и движка.
